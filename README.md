# home-site-hub

该仓库用于归档和发布多个独立的 HTML 页面。这些页面各自独立运作，不关联任何特定域名或网站，主要用于内容展示与存档。

## 项目简介

home-site-hub 是一个轻量化的 HTML 页面集合仓库。你可以在这里找到若干静态页面，它们可能包括个人介绍、项目展示、工具页面或信息索引等。所有页面均为纯前端实现，无需后端服务支持，可直接通过浏览器打开浏览。

## 目录说明

```
home-site-hub/
├── index.html          # 主入口页面，提供页面导航或索引
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **index.html**：仓库首页，通常包含指向各页面的链接或目录。
- **pages/**：所有归档的独立 HTML 页面均放置于此目录下，便于管理。
- **assets/**：存放页面所需的公共静态资源，如样式表、JavaScript 脚本和图片。

## 页面归档说明

本仓库中的每个 HTML 页面都代表一个独立的内容单元。归档遵循以下原则：

- 每个页面自包含，不依赖其他页面资源（公共资源除外）。
- 页面结构清晰，注释完整，便于后续维护或二次开发。
- 所有页面均为静态，可直接通过文件系统或静态托管服务访问。

如需新增页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加相应链接。

## 维护说明

- **更新**：修改现有页面时，请保持原有结构和注释，避免破坏兼容性。
- **新增**：添加新页面时，建议在 `index.html` 中同步更新导航。
- **删除**：移除页面时，请同时清理 `index.html` 中的对应链接。
- **资源管理**：公共资源应统一放在 `assets/` 下，避免重复引入。
